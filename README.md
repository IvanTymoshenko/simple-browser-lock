# Simple Browser Lock

A lightweight, secure, and private browser lock extension for Google Chrome.
To clarify, I am not a coder or a programmer, so 90% of the code for this extention was written by AI.
I just wanted to have a tool that can lock my account from other people at work, and that is how come up with an idea of chrome extention, because Chrome not allowing to do it natively.

### Features
* **Instant Lock:** Hides all open tabs and windows immediately.
* **Privacy First:** No data leaves your computer. Passwords are hashed locally.
* **Dark Mode:** Kinda modern interface.
* **Recovery System:** Includes recovery code system.
* **Smart Restore:** Tabs are restored exactly as you left them upon unlocking.

### How to Install
Install manually using the steps below:

1. Click the green **Code** button above and select **Download ZIP**.
2. Unzip the file to a folder on your computer.
3. Open Chrome and go to `chrome://extensions`.
4. Toggle **Developer mode** on (top right corner).
5. Click **Load unpacked**.
6. Select the folder where you unzipped the files.
7. Done! You will see the setup window pop up.

### Privacy Policy
This extension does not collect, store, or transmit any personal user data to external servers. All passwords, recovery codes, and session data are stored locally on your device using Chrome's local storage API.
